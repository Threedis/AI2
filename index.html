This utility automates the generation of HDFC bulk salary payment files — 275TSAL (HDFC-to-HDFC internal transfers) and 275TNES (NEFT transfers to other banks) — directly from your monthly Bank Advise Excel sheet. No manual formatting, no copy-paste errors.

The tool detects whether both a Salary sheet and an OT sheet are present in the uploaded file. If both are found, 4 files are generated in one run; if only the Salary sheet is present, 2 files are generated. Column order in the Excel does not matter — headers are auto-detected by name.

Standard Operating Procedure
1
Upload the Bank Advise Excel
Drag and drop or click to browse. The file must contain a Salary sheet (and optionally an OT sheet). Supported: .xlsx / .xls
2
Verify auto-detected column mapping
After upload, the tool shows which Excel column was mapped to each field (EMP Code, Account, IFSC, Net Pay, etc.). Confirm before proceeding.
3
Enter payment date and last serial number
Select the payment date (DDMM is auto-derived for filenames). Enter the last serial number used — output files will consume the next 2 or 4 serial slots.
4
Generate and download files
Click Generate Files. Review the discrepancy report (zero-pay exclusions, missing IFSC, blank accounts), then download individually or all at once.
✓
Upload to HDFC Corporate Net Banking
Upload each generated file via Bulk Upload → 275TSAL / 275TNES in the HDFC Corporate portal. Verify the debit confirmation after submission.
